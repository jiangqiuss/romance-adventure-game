# 💕 心动瞬间 · 恋爱冒险

一款互动式恋爱文字冒险游戏，体验浪漫故事、做出选择、开启不同结局。

## 🎮 在线游玩

👉 [点击这里开始游戏](https://jiangqiuss.github.io/romance-adventure-game/)

## ✨ 游戏特色

### 💖 恋爱系统
- 多位可攻略角色
- 分支对话选择
- 多结局系统
- 好感度累积

### 🎭 剧情体验
- 精心编写的恋爱故事
- 沉浸式对话体验
- 情感丰富的角色互动
- 出人意料的剧情转折

### 🎨 视觉设计
- 粒子动效背景
- 角色立绘展示
- 场景切换动画
- 响应式设计

## 🎯 游戏玩法

1. **阅读** - 阅读剧情故事
2. **选择** - 做出对话选择
3. **互动** - 与角色互动
4. **探索** - 解锁不同结局

## 🛠️ 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas 粒子动画
- 无任何外部依赖
- 响应式设计

## 📦 部署

直接将 index.html 托管到任意静态服务器即可。推荐使用 GitHub Pages：

1. Fork 本仓库
2. 进入 Settings → Pages
3. Source 选择 main 分支
4. 访问 https://<username>.github.io/romance-adventure-game/
